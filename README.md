# Random Username Generator

Obtain a random username, or a random username of a max length.

## How to call

Send a GET request to https://random-username-gen.onrender.com/username, with optional maxLen parameter to obtain a random username of size <= maxLen

### Example:

GET https://random-username-gen.onrender.com/username?maxLen=5

**Response**

```json
{
  "username": "Mit34"
}
```
